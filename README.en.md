# home_monitor

#### Description
使用PyTorch实时获取萤石云视频，并使用深度学习对视频进行分析，并将分析结果发送到邮箱。可应用于摄像头实时人物检测，发现人形及时通知，并存储视频。深度学习模型可以自己定义，也可以使用本仓库的，也可以将yolov5的模型应用在本项目下。
